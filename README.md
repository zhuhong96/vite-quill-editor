## Install


```bash
npm i vite-quill-editor --save
```

## Usage

Global introduction

```bash
import QuillEditor from 'vite-quill-editor';
import "vite-quill-editor/index.css";
const app = createApp(App);

app.use(QuillEditor).mount('#app')
```

Component introduction

```bash
import { QuillEditor } from 'vite-quill-editor';
import "vite-quill-editor/index.css";
```

html

```bash
 <quill-editor />
```
